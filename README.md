# Opencyclone2_3
This repository contains the complete documentation and production files for Opencyclone_2.3, an open-source vacuum cleaner developed by Lion Sanguinette and originally initiated together with Jonathan Stein during their Industrial Design studies at Burg Giebichenstein University of Art and Design.

The folders include all files required to understand, reproduce, modify, and further develop the project. These comprise 3D-printable STL files, editable CAD data in STEP format, technical components, preparation files, assembly instructions, and part lists. The documentation is structured to support both individual makers and professional manufacturers.

Opencyclone_2.3 explores how an everyday industrial product can be redesigned as an open, repairable, and adaptable system. By making all relevant design and production information publicly accessible, the project encourages local manufacturing, repair, and collaborative development while demonstrating that open-source hardware can achieve the functionality and quality expected from contemporary consumer products.

The repository is intended not only as a source of manufacturing data, but also as a resource for learning, experimentation, and further contributions to the project.
